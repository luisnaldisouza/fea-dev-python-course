# Curso de Python – FEA.dev 2024

Este repositório reúne as listas de exercícios desenvolvidas durante o Curso de Python da FEA.dev, realizado em 2024. O objetivo é registrar minha evolução prática em programação, desde os fundamentos da linguagem até os primeiros passos em análise e visualização de dados.

O curso teve uma proposta introdutória e aplicada, conectando lógica de programação com problemas próximos do uso acadêmico e profissional. Ao longo das listas, foram trabalhados temas como variáveis, operadores, condicionais, loops, estruturas de dados, funções, NumPy, Pandas e bibliotecas de visualização.

## Conteúdos estudados

| Lista | Tema principal | Conteúdos praticados |
|---|---|---|
| 01 | Fundamentos de Python | Variáveis, operadores, tipos de dados, entrada de dados e manipulação básica de strings |
| 02 | Condicionais e loops | Estruturas condicionais, `while`, validações e problemas com repetição |
| 03 | Estruturas de dados | Listas, tuplas, conjuntos, dicionários e introdução ao `for` |
| 04 | Funções | Criação de funções, parâmetros, retorno de valores e organização do código |
| 05 | NumPy | Arrays, operações numéricas, matrizes e cálculos com dados |
| 06 | Pandas I | Leitura de bases, DataFrames, seleção, filtros e análise inicial de dados |
| 07 | Pandas II | `apply`, datas, agrupamentos, tabelas dinâmicas e junção de tabelas |
| 08 | Visualização de dados | Gráficos com Matplotlib e Seaborn, análise visual e interpretação de dados |

## Como executar

1. Clone este repositório:

```bash
git clone https://github.com/SEU-USUARIO/NOME-DO-REPOSITORIO.git
```

2. Acesse a pasta do projeto:

```bash
cd NOME-DO-REPOSITORIO
```

3. Instale as dependências:

```bash
pip install -r requirements.txt
```

4. Abra os notebooks:

```bash
jupyter notebook
```
