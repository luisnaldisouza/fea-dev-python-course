# Projeto Final – Análise de Dados Educacionais

Este projeto foi desenvolvido como entrega final do Curso de Python da FEA.dev 2024. A proposta foi aplicar os principais conteúdos estudados ao longo do curso em uma análise exploratória de dados educacionais.

## Objetivo

A pergunta norteadora do projeto foi:

> Quais fatores demográficos, acadêmicos e comportamentais mais influenciam a classificação e a média de notas dos estudantes?

A análise utiliza uma base de dados de estudantes e investiga relações entre desempenho acadêmico e variáveis como idade, gênero, etnia, horas de estudo semanais, faltas, aulas particulares, apoio dos pais e participação em atividades extracurriculares.

## Principais etapas

- Importação e leitura da base de dados.
- Renomeação de colunas para facilitar a interpretação.
- Análise inicial da estrutura da base.
- Criação de classificações auxiliares.
- Agrupamentos com Pandas.
- Visualizações com Matplotlib e Seaborn.
- Interpretação dos fatores associados ao desempenho dos estudantes.

## Bibliotecas utilizadas

- Pandas
- NumPy
- Matplotlib
- Seaborn

## Observação

Este projeto tem finalidade acadêmica e de portfólio, mostrando a aplicação prática dos conteúdos estudados no curso.
